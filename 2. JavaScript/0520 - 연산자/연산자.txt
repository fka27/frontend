1. 산술 연산자
    : 사칙연산 및 기본적인 연산을 하는 연산자

    + : 덧셈
    - : 뺄셈
    * : 곱셈
    / : 나눗셈
    % : 나눈 나머지 값
    ** : 제곱


2. 비교 연산자
    :피연산자 사이의 상대적인 크기를 판단하여 참 또는 거짓을 반환

    > : 크다
    < : 작다
    >= : 크거나 같다
    <= : 작거나 같다
    == : 같다(타입이 달라도 값이 같으면 참)
    === : 같다(타입과 값이 같아야 참)
    != :다르다(타입이 다르면 값이 같아도 거짓)
    !== : 다르다(타입이 달라도 값이 같으면 참)


3. 대입 연산자
    :변수나 상수에 값을 대입할 때 사용하는 연산자

    let aaa = 1000
        :우측의 값을 왼쪽(변수 또는 상수)에 대입
    result = result + 100  =>  result += 100
    result = result - 100  =>  result -= 100
    result = result * 100  =>  result *= 100
    result = result % 100  =>  result %= 100


4. 증감 연산자
    :1씩 값을 증가시키거나 또는 감소 시킬 때 사용하는 연산자
    연산자의 위치가 어디에 있는지에 따라 결과가 달라질 수 있음

    ++변수 : 변수의 값을 1증가 시킨 후, 다음 연산 진행
    변수++ : 먼저 연산 후, 변수의 값을 1증가
    --변수 : 변수의 값을 1감소 시킨 후, 다음 연산 진행
    변수-- : 먼저 연산 후, 변수의 값을 1감소


5. 논리 연산자
    :주어진 논리식을 판단하여 참과 거짓을 반환

    and(&&)연산 : 두 논리식 모두 참이면 참을 반환
    or(||)연산 : 두 논리식 중 하나라도 참이면 참을 반환    
    not(!)연산 : 논리식의 결과가 참이면 거짓, 거짓이면 참으로 반환


6. 삼항 연산자
    :조건식에 따른 참, 거짓에 따라 반환값이 달라지는 연산자

                           true : false
    변수(상수) = 조건식 ? 결과1 : 결과2
    (-> 결과가 true일 때 결과1 실행/ false일 때 결과2 실행)

    변수(상수) = 조건식 && 참일 때 결과1 실행
